# `00` Bienvenida

Bienvenido a **Interview Coding Challenges**.

Este proyecto contiene ejercicios practicos de programacion para prepararte para entrevistas tecnicas, desde problemas de calentamiento hasta patrones comunes de entrevista.

## Como avanzar en los desafios

1. Abre cada carpeta de ejercicio en orden y lee sus instrucciones.
2. Escribe tu solucion en el `app.js` correspondiente.
3. Usa la opcion **Execute** cuando necesites correr el codigo y revisar la salida.
4. Ejecuta **Test** para validar tu solucion.
5. Pasa al siguiente reto cuando los tests esten en verde.

Enfocate primero en una solucion correcta y clara. Luego mejora legibilidad y eficiencia como lo harias en una entrevista real.
