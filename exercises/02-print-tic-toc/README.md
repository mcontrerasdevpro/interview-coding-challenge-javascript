# `02` TicToc

Write a program that prints numbers from 1 to 100 and:

- For multiples of `3`, print `"Tic"` instead of the number.
- For multiples of `5`, print `"Toc"`.
- For numbers divisible by both 3 and 5, print `"TicToc"`.

Example output:

```sh
1
2
Tic
4
Toc
...
14
TicToc
```

Explanation: the program iterates from 1 to 100. It prints numbers by default, replaces multiples of 3 with `Tic`, multiples of 5 with `Toc`, and multiples of both with `TicToc`.
