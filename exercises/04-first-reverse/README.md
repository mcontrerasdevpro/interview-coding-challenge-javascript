# `04` First Reverse

Have the function FirstReverse(str) take the str parameter being passed and return the string in reversed order. For example: if the input string is "Hello World and Coders" then your program should return the string sredoC dna dlroW olleH.

## Examples

```
Input: "coderbyte"
Output: etybredoc
```
Explanation: each character order is inverted, so the first character becomes the last.

```
Input: "I Love Code"
Output: edoC evoL I
```
Explanation: spaces are preserved, but the full string is reversed character by character.
